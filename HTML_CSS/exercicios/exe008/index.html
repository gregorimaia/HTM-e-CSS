Texto Marcado

Podemos marcar um texto com a Tag mark.

Podemos tambem alterar a cor do marcador de texto com a Tag mark e adicionando um style dentro da Tag mark. Mark é uma Tag HTLM enquanto Style ja é mais CSS.

Mudando a cor de várias marcações ao mesmo tempo

Se no meu texto houver milhares de marcações e eu quiser que todas tenham a mesma cor, a Tag mark deve ser acrescentada no head e assim todas as marcações da mesma página serão iguais.

Testo Grande e Pequeno

Utilizando a Tag big para textos grandes, mas observe que big está em vermelho por ser uma Tag obsoleta, pois não tem sentido fazer um texto com letras um pouco maiores do que o normal.

Para textos em letras pequenas utilizamos a Tag small, essa Tag já não é obsoleta, pois para citações ou outros motivos já se utiliza textos em letras pequenas.

Texto deletado

Podemos marcar um texto como excluído para indicar que deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que foi adicionado depois.

Temos o texto sublinhado com a Tag u que é não semântica.

Texto sobrescrito

Para adicionarmos potências como 42 utilizamos a Tag sup.

Texto subescrito

Pra adicionarmos subescritos como em fórmulas químicas H2O utilizamos a Tag sub.
